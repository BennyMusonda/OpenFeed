##  Core Features
*   *Dynamic Navigation Bar:* Fully structured top nav including search spacing, profile layouts, and navigation tabs.
*   *Interactive Controls:* Styled implementation of the 'Live video', 'Photo/video', and 'Feeling/activity' action buttons.
*   *Stories Section:* A responsive flexbox/grid wrapper displaying the classic user story cards.
*   *Feed & Post Layout:* Clean card design for news feed posts, featuring user headers, image placeholders, and reaction bars.

##  Built With
*   *HTML5:* Semantic structural layout.
*   *CSS3:* Custom styling, Flexbox layouts for positioning, and responsive adjustments.
*   *JavaScript:* For user Interactions.
  
##  What I Practiced
*   Mastering component alignment using CSS Flexbox.
*   Replicating professional UI/UX spacing, padding, and hover transitions.
*   Keeping code clean and scannable.
